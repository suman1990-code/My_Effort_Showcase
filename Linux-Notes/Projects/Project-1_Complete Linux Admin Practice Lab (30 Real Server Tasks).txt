Project-1: Complete Linux Admin Practice Lab (30 Real Server Tasks)

	1. Use & Group Management
		Scenario: A new employee joined the company.
		
		Task: 
		a. Create User : Suman.Mondal
		b. Home Directory : /data/suman.mondal
		c. Shell : bin/bash
		
		Commands used:
			   a. sudo useradd suman.mondal -m -d /data/suman.mondal -s /bin/bash suman.mondal
					Explanation:
						Option						Meaning
						-m							Create home directory
						-d /data/suman.mondal		Custom home directory
						-s /bin/bash 				Default login shell
						suman.mondal				User Name
			   b. sudo passwd suman.mondal
			   c. id suman.mondal
		
		Output:
			[sumancse1990@mylab ~]$ 
			[sumancse1990@mylab ~]$ sudo useradd  -m -d /data/suman.mondal -s /bin/bash suman.mondal

			[sumancse1990@mylab ~]$ sudo passwd suman.mondal
